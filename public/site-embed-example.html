TakeShape Site Embed

This example demonstrates how to embed the TakeShape chat and session tracking functionality across your entire website.

How It Works

The TakeShape Site Embed script adds session tracking to your website and optionally includes a chat widget. This allows you to:

  • Track user sessions to understand visitor behavior
  • Engage with visitors through an optional chat interface anywhere on your site
  • Collect leads by directing users to your TakeShape contact form

The script is designed to be lightweight and non-intrusive, with customization options to match your brand.

Installation

To add the TakeShape Site Embed to your website, simply add the following script tag to your HTML:

<script 
  src="https://your-takeshape-domain.com/site-embed.js" 
  data-painter-id="YOUR_PAINTER_ID"
  data-enable-chat="true"
  data-primary-color="#ff385c"
  data-font-family="Arial, sans-serif"
  data-position="bottom-right">
</script>

Replace YOUR_PAINTER_ID with your actual painter ID from TakeShape.

Customization Options

You can customize the appearance and behavior of the embed using data attributes:

Attribute Description Default
data-painter-id Your painter ID from TakeShape (required) None
data-enable-chat Whether to enable the chat widget false
data-primary-color Primary color for buttons and accents #ff385c
data-font-family Font family for the chat widget system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Oxygen, Ubuntu, Cantarell, "Open Sans", "Helvetica Neue", sans-serif
data-position Position of the chat widget (bottom-right, bottom-left, top-right, top-left) bottom-right
data-text-color Text color for the chat widget #333333
data-background-color Background color for the chat panel #ffffff
