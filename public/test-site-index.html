Testing with URL Parameters

You can customize the embed scripts using URL parameters. Here are some examples:

Parameter Description Example
painterId Sets the painter ID for the site-wide embed script Try it
enableChat Enables or disables the chat widget (true/false) Try it
primaryColor Sets the primary color for the embed scripts (hex code) Try it
position Sets the position of the chat widget Try it

You can combine multiple parameters:

/test-site/index.html?painterId=YOUR_PAINTER_ID&enableChat=true&primaryColor=%23ff0000&position=bottom-left
