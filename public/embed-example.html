TakeShape Room Scanner Embed Example

This example demonstrates how to embed the TakeShape Room Scanner into your website. The embed can be customized to match your brand colors and style.

Customize the Embed

Use the form below to customize the appearance of the embed. Changes will be applied when you click "Apply Changes".

8px

HTML Code Example

<iframe 
  src="/embed/demo?primaryColor=%23ff385c&secondaryColor=%23007aff&backgroundColor=%23ffffff&textColor=%23333333&fontFamily=Arial%2C%20sans-serif&borderRadius=8&enableChat=false" 
  width="100%" 
  height="700" 
  style="border: none; border-radius: 8px;" 
  allowfullscreen
></iframe>

JavaScript Initialization Example

<div id="takeshape-embed-container"></div>

<script src="/embed.js"></script>
<script>
  TakeShapeEmbed.init({
    container: 'takeshape-embed-container',
    primaryColor: '#ff385c',
    secondaryColor: '#007aff',
    backgroundColor: '#ffffff',
    textColor: '#333333',
    fontFamily: 'Arial, sans-serif',
    borderRadius: '8px',
    enableChat: false
  });
</script>
